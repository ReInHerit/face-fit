In order to send mails with morphed results you have to use (or create a new email with) gmail.
Then you have to generate an App Password following the istructions of the first point on this link:
https://www.interviewqs.com/blog/py-email
Then feed the password_gmail.json file with email and the generated password.
You can find two version of the json, one for the Desktop App and one for the Web App:
- the first in facefit_python folder
- the second in facefit_javascript/public/json folder
